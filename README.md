# KiBi on Steroids!

##### Description
Alpine Docker container w/ Elasticsearch 5.x + Kibi 5.x + Sentinl 5.x

##### Run
```
docker run -d -p 9200:9200 -p 5606:5606 qxip/docker-kibi-steroids
```
